# SAP BASIS Daily Operations Runbook

## Overview

This repository contains daily operational and monitoring activities performed by SAP BASIS administrators to ensure SAP system health, stability, and performance.

---

## Daily Health Checks

### SM51 - Application Server Monitoring
- Verify active application servers
- Check server availability
- Monitor instance status

### SM50 - Work Process Monitoring
- Monitor running work processes
- Identify long-running processes
- Analyze system workload

### SM66 - Global Work Process Monitoring
- View work processes across all servers
- Detect performance bottlenecks
- Monitor resource utilization

---

## Background Job Monitoring

### SM37
- Monitor scheduled jobs
- Check cancelled jobs
- Analyze job logs
- Verify critical batch processing

---

## System Log Analysis

### SM21
- Review system logs
- Identify warnings and errors
- Monitor system events

### ST22
- Analyze ABAP runtime dumps
- Review error details
- Perform root cause analysis

---

## Database Monitoring

### DBACOCKPIT
- Database health checks
- Backup monitoring
- Performance monitoring
- Space management

---

## ICM Monitoring

### SMICM
- Monitor ICM services
- Check HTTP/HTTPS connectivity
- Review ICM traces and logs
